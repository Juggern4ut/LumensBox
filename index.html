Lumens Lightbox

This is the Demo-Page for the Lumens Lightbox. Please keep in mind, that you should never place that many lightboxes in a single page, but rather change the content of one globally used Lightbox. This is just for demonstration purposes and if you spam the buttons to fast or press Esc while opening a box and open it right again, it might cause problems. In a running environment, this should never be a problem.
